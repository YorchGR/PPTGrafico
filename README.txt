# PPTGrafico
Juego Piedra Papel TIjera

Programa de escritorio, basado en el juego de de piedra papel o tijera, creado con la intención de poner en práctica lo aprendido en el curso (FP superior DAM) durante el primer 
año. También lo use para profundizar en diferentes aspectos como fueron, el modo gráfico, la utilización de sonido o la de animaciones.

La aplicación solo cuenta con un modo de juego, jugador contra cpu.
